# System Requirements

TCAdmin 3 runs as a standalone .NET 9 application. System requirements are based on .NET 9 supported operating systems.

### Dependencies

- A MySQL database is required. A database located on the master server is recommended for best performance. [MariaDB](https://mariadb.org/download)
#### Required by some games

- [Microsoft Visual C++ **(x64)**](https://aka.ms/vs/17/release/vc_redist.x64.exe) 2015 to 2022
- [Microsoft Visual C++ **(x86)**](https://aka.ms/vs/17/release/vc_redist.x86.exe) 2015 to 2022
- [Direct X](https://www.microsoft.com/en-us/download/details.aspx?id=35&6B49FDFB-8E5B-4B07-BC31-15695C5A2143=1)
- Java **required for Minecraft.** We recommend [Adoptium variant](https://adoptium.net/)

#### Optional

- [Docker Desktop](https://docs.docker.com/desktop/setup/install/windows-install/) or [Docker Engine](https://docs.docker.com/engine/install/) - Required for running containerized services

### Dependencies

- A MySQL database is required. A database located on the master server is recommended for best performance. [MariaDB](https://mariadb.org/download)
:::warning
Prior knowledge of Linux server management is required. TCAdmin support does not provide instruction on tasks such as logging in to your server using Putty, executing commands, or installing dependencies for game/voice servers or tools.
:::

#### Required by some games

- Install 32bit libraries for steamcmd and 32bit games: [SteamCmd Libraries](https://developer.valvesoftware.com/wiki/SteamCMD#32-Bit_Libraries_on_64-Bit_Linux_Systems)
- Java **required for Minecraft.** We recommend [Adoptium variant](https://adoptium.net/)

#### Optional

- [Docker Engine](https://docs.docker.com/engine/install/) - Required for running containerized services
