# WHMCS Module

A WHMCS provisioning module for TCAdmin 3. It creates and manages game servers in
TCAdmin directly from WHMCS — provisioning on order; suspending, unsuspending, and
terminating in step with billing; password changes; package upgrades; Start / Stop /
Restart / Force-Kill controls in the client area; and one-click login to the TCAdmin
control panel.

:::tip[Download]

[**Download the WHMCS module — v1.0.0**](https://github.com/TotalControlAdmin/TCA3BillingModules/releases/tag/whmcs-v1.0.0)

:::



For full installation, configuration, and API-key permission setup, see the
[module README on GitHub](https://github.com/TotalControlAdmin/TCA3BillingModules/blob/master/WHMCS/README.md).
